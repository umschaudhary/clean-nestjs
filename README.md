## Project Setup Instructions

1. Update `.env` file
2. Run project with docker
```bash 
docker-compose up 
```
3. Run migrations
```bash
docker-compose run web bash
npx prisma migrate dev
```
4. Access api running on [Localhost](http://localhost:3000)
5. Swagger docs [Link](http://localhost:3000/api)
